# eshopping

eshopping 是一个基于Web的电子商务平台，旨在为用户提供一个方便快捷的在线购物体验。该平台包括了商品展示、购物车、结算、订单管理等功能。

## 开始使用

以下说明可以帮助您将本项目部署到本地环境并进行开发和测试。

### 前置条件

为了运行本项目，您需要：

- Node.js
- MongoDB

### 安装

1. 克隆本仓库：

```
git clone https://github.com/xingnan88/eshopping.git
```

2. 安装依赖：

```
npm install
```

3. 启动服务器：

```
npm start
```

4. 访问 `http://localhost:3000` 即可进入应用。

## 功能

eshopping 包括以下功能：

- 商品展示
- 购物车
- 结算
- 订单管理
- 用户管理

## 技术栈

- Node.js
- Express.js
- MongoDB
- Bootstrap
